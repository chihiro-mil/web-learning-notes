# CUIを使ったディレクトリ（ファイル）操作方法

## 3つの特別なディレクトリ
- カレントディレクトリ：現在地のディレクトリ。パスは「.」
- ホームディレクトリ：GitBashなどを起動した時の初期表示されるディレクトリ。パスは「~」
- ルートディレクトリ：ファイルシステムの一番上の階層。パスは「/」

## 絶対パス
ルートディレクトリを起点として移動したいディレクトリまですべてパスを使用
例）/c/Users

## 相対パス
カレントディレクトリを起点として「..」などのパスを使用する
例） 1つ上の階層に移動：..　2つ上の階層に移動：../..　 
ルートディレクトリから下の階層に移動する時：c/ や　c/Users など現在地から下の階層パスを指定

## ディレクトリを操作する基本コマンド
- pwd：カレントディレクトリの絶対パス表示
- ls：ディレクトリの内容を表示
- cd：ディレクトリの移動
- mkdir：ディレクトリ作成
- touch：ファイル作成
